# Gestion des Inscriptions Scolaires

Bienvenue dans le projet de gestion des inscriptions scolaires. Ce projet vise à créer une application web permettant aux établissements scolaires de gérer efficacement les inscriptions des nouveaux étudiants.

## Introduction

Ce projet a pour but de simplifier le processus d'inscription des étudiants, en fournissant une plateforme conviviale pour le personnel administratif et les nouveaux étudiants.

## Fonctionnalités de Base

- Authentification des utilisateurs
- Page d'inscription pour les nouveaux étudiants
- Gestion des informations personnelles des étudiants
- ...

## Fonctionnalités Avancées

- Planification des emplois du temps pour les tests (écrit, oral)
- Système de gestion des documents
- Système de feedback
- Module d'orientation virtuelle
- ...

## Environnement de Développement

- Backend : Laravel, PHP, SQL
- Frontend: HTML, SCSS, Typescript, Angular
